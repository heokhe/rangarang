# 🌈 Rangarang
Rangarang (Persian word for "Colorful") is a tiny library that **chooses the best colors from a picture**.

It is:
- Very, very lightweight ⚡️
- Fast 🤔
- TypeScript-friendly ✅
- Works in both Node.js and broswer environments 🌍
